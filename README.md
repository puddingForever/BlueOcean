## 오픈API 활용 
-미항공우주국(NASA)에서 제공되는 다양한 json형식의 오픈 API를 이용하여 클라이언트가 우주 정보에 쉽게 접근할 수 있도록 구현 <br>
-화성탐사로봇의 API를 이용하여 화성의 사진을 보여줄 수 있도록 구현 
-클라이언트는 화성의 날을 선택하여 원하는 날짜의 사진을 볼 수 있음
