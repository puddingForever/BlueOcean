# 우주 여행 프로젝트 Blue Ocean

# 프로젝트 구상

### 프로젝트 기간
2023-05-01~

###기술스택
Java, Spring5,OracleDB,JSP

### 핵심기능 
#### 회원관리
+ 회원가입 및 로그인처리 
+ 회원프로필
#### 쇼핑몰 기능
+ 예약하기 및 예약페이지 
#### 게시판 페이지
+ 후기게시판 및 댓글 관리
#### NASA 오픈 API 활용 
+  화성에 있는 로봇이 촬영한 화성사진 
+  NASA가 선정한 오늘의 우주사진
+  회원프로필에 있는 갤러리에 API로부터 받아온 사진 추가 가능
#### 다국어 처리기능
+ 한국어,영어,독일어
#### D-DAY 이벤트 처리 
+ 팝업창으로 D-DAY 이벤트 설정
+ 로그인사용자는 응모기능
